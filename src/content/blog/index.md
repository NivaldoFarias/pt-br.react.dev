<BlogCard title="React Canaries: Incremental Feature Rollout Outside Meta" date="May 3, 2023" url="/blog/2023/05/03/react-canaries">

Tradicionalmente, os novos recursos do React costumavam estar disponíveis apenas no Meta primeiro e, posteriormente, nas versões de código aberto. Gostaríamos de oferecer à comunidade React uma opção para adotar novos recursos individuais assim que seu design estiver próximo do final - semelhante a como o Meta usa o React internamente. Estamos introduzindo um novo canal de lançamento Canary com suporte oficial. Ele permite que configurações selecionadas como frameworks desacoplem a adoção de recursos individuais do React da programação de lançamento do React.

</BlogCard>
