<Intro>

O desenvolvimento do React é liderado por uma equipe dedicada que trabalha em tempo integral na Meta. Ele também recebe contribuições de pessoas de todo o mundo.

</Intro>

## React Core {/*react-core*/}

Os membros da equipe React Core trabalham em tempo integral nas APIs dos componentes principais, no motor que impulsiona o React DOM e o React Native, no React DevTools e no site de documentação do React.
