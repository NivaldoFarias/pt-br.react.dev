## Como contribuir {/*how-to-contribute*/}

Você pode contribuir para os esforços de tradução!

A comunidade conduz o trabalho de tradução para a documentação do React em cada fork específico do idioma de react.dev. O trabalho de tradução típico envolve traduzir diretamente um arquivo Markdown e criar um pull request. Clique no link "contribuir" acima para o repositório GitHub do seu idioma e siga as instruções lá para ajudar no esforço de tradução.

Se você deseja iniciar uma nova tradução para o seu idioma, visite: [translations.react.dev](https://github.com/reactjs/translations.react.dev)
