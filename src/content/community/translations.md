## Como contribuir {/*how-to-contribute*/}

Você pode contribuir para os esforços de tradução!

A comunidade realiza o trabalho de tradução dos documentos do React em cada fork específico por idioma do react.dev. O trabalho de tradução típico envolve a tradução direta de um arquivo Markdown e a criação de um pull request. Clique no link "contribuir" acima para o repositório do GitHub para o seu idioma e siga as instruções lá para ajudar no esforço de tradução.

Se você deseja iniciar uma nova tradução para o seu idioma, visite: [translations.react.dev](https://github.com/reactjs/translations.react.dev)
