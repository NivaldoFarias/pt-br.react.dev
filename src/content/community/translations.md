## Como contribuir {/*how-to-contribute*/}

Você pode contribuir para os esforços de tradução!

A comunidade conduz o trabalho de tradução para os documentos do React em cada fork do react.dev específico do idioma. O trabalho de tradução típico envolve a tradução direta de um arquivo Markdown e a criação de um pull request. Clique no link "contribuir" acima para o repositório do GitHub para o seu idioma e siga as instruções lá para ajudar com o esforço de tradução.

Se você quiser iniciar uma nova tradução para seu idioma, visite: [translations.react.dev](https://github.com/reactjs/translations.react.dev)
