<Intro>

Na build de produção minificada do React, evitamos enviar mensagens de erro completas para reduzir o número de bytes enviados pela rede.

</Intro>

Recomendamos fortemente o uso da build de desenvolvimento localmente ao depurar seu aplicativo, pois ela rastreia informações de depuração adicionais e fornece avisos úteis sobre possíveis problemas em seus aplicativos, mas se você encontrar uma exceção ao usar a build de produção, esta página remontará a mensagem de erro original.

O texto completo do erro que você acabou de encontrar é:

<ErrorDecoder />

Este erro ocorre quando você passa um valor BigInt de um Componente de Servidor para um Componente de Cliente.
